# RandomItems

A mod created by [apixon](https://github.com/Apix0n) adding everything I want to add to Minecraft like dimensions, new blocks and items.

## How to download

Go to https://curseforge.com/minecraft/mc-mods/randomitems/files

## Which version should I choose?

[RandomItems](https://curseforge.com/minecraft/mc-mods/randomitems) is based on Forge's 1.12.2 version.

Download Forge 1.12.2 [here](http://files.minecraftforge.net/maven/net/minecraftforge/forge/index_1.12.2.html)
